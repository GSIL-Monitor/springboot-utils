   # 新浪邮箱配置
   spring:
     mail:
       host: smtp.exmail.qq.com
       username: chenmingjian@beeseven.com
       password: aA2824406956
       default-encoding: UTF-8

   # QQ邮箱配置
   spring:
     mail:
       host: smtp.qq.com
       username: chenmingjian@beeseven.com
       password: aA2824406956  # 不是邮箱密码，而是授权码
       default-encoding: UTF-8

    # 网易系(126/163/yeah)邮箱配置
    spring:
      mail:
        host: smtp.126.com
        username: chenmingjian@beeseven.com
        password: aA2824406956
        default-encoding: UTF-8

除了qq邮箱发件密码使用的是qq邮箱提供的授权码之外，别的邮箱都是直接使用邮箱本密码
